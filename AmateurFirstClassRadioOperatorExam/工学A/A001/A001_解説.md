# A001：AM(A3E)送信機の変調度・搬送波電力・平均電力の計算

## 出題頻度

- ★★★★★(5)

## 学習のポイント

AM（A3E）送信機では、変調をかけることで搬送波の両側にサイドバンドが生じ、総送信電力が増加します。この単元では以下の理解が求められます。

- 変調度 $m$ と搬送波電力 $P_C$ から**被変調波の平均電力**$P$を求める公式を正確に覚える
- 平均電力と搬送波電力が与えられたときに**変調度を逆算**できる
- 搬送波の振幅値（最大値）から**変調波の実効電圧**を計算できる

アマチュア無線技士として、送信機の変調状態を数値で把握し、適切な電力管理ができるようになることが目標です。

---

## 問題の攻略方法

**最重要公式をまず覚える**

$$P = \left(1 + \dfrac{m^2}{2}\right)P_C$$
$$P_C：搬送波の電力$$
$$P：AM(A3E波)の平均電力$$

- $m$ は変調度（百分率 [%] で与えられたら **必ず100で割って小数**に直す）
- **「無変調時の電力」＝「搬送波電力 $P_C$」** と読み替える
- 変調度を逆算する問題では $P/P_C$ を先に計算し、$1$ を引いて $2$ を掛けて平方根を取る
- 電圧の実効値問題は「振幅 ÷ √2」が基本。AM波では係数 $\sqrt{1 + m^2/2}$ を掛ける

---

## 詳しい解説

### AM（A3E）波の構造

変調度 $m$（$0 \leq m \leq 1$）を用いると、AM波の瞬時電圧は次のように表されます。

$$v(t) = A_C\left(1 + m\cos\omega_s t\right)\cos\omega_C t$$
$$\omega_C：搬送波$$
$$\omega_s：信号波$$

これらの式を展開すると、**搬送波・上側サイドバンド・下側サイドバンド**の3成分に分解できます。
（展開方法は一陸技レベルです。三角関数の積和公式を使いますので、ここでは結果のみ使用します）

| 成分 | 振幅 |
|------|------|
| 搬送波 | $A_C$ |
| 上側サイドバンド | $\dfrac{m A_C}{2}$ |
| 下側サイドバンド | $\dfrac{m A_C}{2}$ |

### 平均電力の公式

平均電力は、各電圧成分の二乗を合計した値になります。また、搬送波電力Pcは、搬送波電圧Acの二乗になります。

$$ P_C = {A_C^2}$$

それらを合計すると：

$$P = P_C + \dfrac{m^2}{4}P_C + \dfrac{m^2}{4}P_C = \left(1 + \dfrac{m^2}{2}\right)P_C$$

> **公式の意味：** 変調をかけても搬送波電力 $P_C$ 自体は変化しません。上下サイドバンド2本分の電力 $\dfrac{m^2}{2}P_C$ が上乗せされます。変調度100%（$m=1$）でも増加分は最大50%です。

### 変調度の逆算

$$m = \sqrt{2\left(\dfrac{P}{P_C} - 1\right)}$$

### AM波の実効電圧

搬送波の振幅（最大値）を $A_C$ とすると：

$$V_{rms} = \dfrac{A_C}{\sqrt{2}}\sqrt{1 + \dfrac{m^2}{2}}$$

## 演習問題

---

**問題 1　公式の選択**

AM（A3E）波の平均電力 $P$ を表す式として、正しいものを選べ。ただし、変調信号は単一正弦波とし、搬送波の平均電力を $P_C$ [W]、変調度を $m \times 100$ [%] とする。

```
1.  P = (1 + m/2) Pc
2.  P = (1 + m²/2) Pc
3.  P = (1 − m²/2) Pc
4.  P = (m/2) Pc
5.  P = (m²/2) Pc
```

**解法：** AM波を3成分に分解すると、搬送波電力 $P_C$ に加え、上下各サイドバンドの電力 $\dfrac{m^2}{4}P_C$ が2本分追加されます。

$$P = P_C + 2 \times \dfrac{m^2}{4}P_C = \left(1 + \dfrac{m^2}{2}\right)P_C$$

選択肢3は符号が「$-$」で誤り。選択肢4・5は搬送波成分が含まれておらず不完全です。

**✅ 正解：2**

---

**問題 2　変調度の逆算 ①**

AM（A3E）送信機において、無変調時の電力が 100 [W] の搬送波を単一の正弦波信号で変調したとき、被変調波の平均電力は 132 [W] であった。このときの変調度として、最も近いものを選べ。

```
1. 50 [%]    2. 60 [%]    3. 70 [%]    4. 80 [%]    5. 90 [%]
```

**解法：**

$$m = \sqrt{2\left(\dfrac{132}{100} - 1\right)} = \sqrt{2 \times 0.32} = \sqrt{0.64} = 0.8$$

$$m × 100 = 80  $$

**✅ 正解：4（80 [%]）**

---

**問題 3　変調後の送信電力の計算**

AM（A3E）送信機において、変調をかけないときの送信電力が 750 [W] であった。単一正弦波で変調度 80 [%] の変調をかけたときの送信電力として、最も近いものを選べ。

```
1. 770 [W]    2. 880 [W]    3. 990 [W]    4. 1,010 [W]    5. 1,100 [W]
```

**解法：**

$$P = \left(1 + \dfrac{0.8^2}{2}\right) \times 750 = 1.32 \times 750 = 990 \text{ [W]}$$

**✅ 正解：3（990 [W]）**

---

**問題 4　搬送波電力の逆算**

AM（A3E）送信機において、被変調波の平均電力は 450 [W]、変調度は 80 [%] であった。無変調のときの搬送波電力として、最も近いものを選べ。

```
1. 360 [W]    2. 340 [W]    3. 320 [W]    4. 300 [W]    5. 280 [W]
```

**解法：**

$$P_C = \dfrac{450}{1 + \dfrac{0.8^2}{2}} = \dfrac{450}{1.32} \approx 340.9 \text{ [W]}$$

**✅ 正解：2（340 [W]）**

---

**問題 5　変調波の実効電圧**

AM（A3E）送信機の出力端子において、変調をかけないときの搬送波電圧の振幅値（最大値）が 100 [V] であった。変調度 50 [%] で変調をかけたときの変調波電圧の実効値として正しいものを選べ。

```
1. 65 [V]    2. 70 [V]    3. 75 [V]    4. 80 [V]    5. 85 [V]
```

**解法：**

$$V_{rms} = \dfrac{100}{\sqrt{2}}\sqrt{1 + \dfrac{0.5^2}{2}} = \dfrac{100}{\sqrt{2}}\sqrt{1.125} = 70.71 \times 1.0607 \approx 75.0 \text{ [V]}$$

**✅ 正解：3（75 [V]）**

---

**問題 6　変調度の逆算 ②**

AM（A3E）送信機において、無変調時の電力が 200 [W] の搬送波を単一の正弦波信号で変調したとき、被変調波の平均電力は 280 [W] であった。このときの変調度として、最も近いものを選べ。

```
1. 75 [%]    2. 80 [%]    3. 85 [%]    4. 90 [%]    5. 95 [%]
```

**解法：**

$$m = \sqrt{2\left(\dfrac{280}{200} - 1\right)} = \sqrt{2 \times 0.4} = \sqrt{0.8} \approx 0.894$$

最も近い値は  90[%]。

**✅ 正解：4（90 [%]）**
