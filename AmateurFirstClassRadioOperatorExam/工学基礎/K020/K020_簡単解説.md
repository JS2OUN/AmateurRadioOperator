# K020：電気と磁気に関する法則および各種電気現象

## 出題頻度

- ★★★★★(5)

## 学習のポイント

この問題は、電気と磁気、そしてさまざまな物理現象の「一番基本となるルール」を正しく理解しているかを確かめるものです。

すでに2アマの知識があるので、公式や言葉は聞いたことがあるかと思います。1アマに向けて大切なのは、それぞれの現象が「何がきっかけ（原因）で、どんな結果が起きるのか」というつながりを、感覚的にすっきりと整理することです。ここをあいまいにせず、パッとイメージできるようになると、アンテナの仕組みや最先端の半導体の動きも、驚くほど簡単に理解できるようになります。

## 問題の攻略方法

試験問題では、法則の名前と説明をわざとあべこべにした「ひっかけ問題」がよく出ます。以下の表を使って、「何から何が生まれるか」という組み合わせをシンプルに整理しておきましょう。

| 法則・現象名 | 説明（きっかけ $\rightarrow$ 起きる結果） |
| --- | --- |
| **アンペアの右ねじの法則** | 電流を流す $\rightarrow$ まわりに **磁界（磁石の力）** ができる。 |
| **フレミングの左手の法則** | 磁石の中で電流を流す $\rightarrow$  **「力」** が生まれて動く。（モーターの仕組み） |
| **フレミングの右手の法則** | 磁石の中で導体を動かす $\rightarrow$  **「電気（電気を動かす力）」** が生まれる。（発電機の仕組み） |
| **レンツの法則** | 電磁誘導の **「向き」** のルール。変化を「邪魔する方向」に電気が流れる。 |
| **ファラデーの法則** | 電磁誘導の **「大きさ」** のルール。磁石の動きが速いほど、生まれる電気が「大きくなる」。 問題文では、**鎖交（さこう）** というキーワードに注目。|
| **ゼーベック効果** | ２つの金属をくっつけて**温度差**をつける $\rightarrow$ **電気**が生まれる。 |
| **ペルチェ効果** | ２つの金属をくっつけて**電流**を流す $\rightarrow$ 片方が熱くなり、片方が冷たくなる（**熱の移動**）。 |
| **トムソン効果** | **１本の金属線**に温度差をつけて電流を流す $\rightarrow$ 熱が出たり吸い取られたりする。 |
| **ホール効果** | 電流に**磁石の力**を横からかける $\rightarrow$ 横に電気がたまって**電圧**ができる。 |
| **圧電効果（ピエゾ効果）** | 結晶を**おす・ひっぱる（力をかける）** $\rightarrow$ **電気**が生まれる。 |
| **磁気ひずみ現象** | 磁石の力をかける $\rightarrow$  **形が縮んだり伸びたり（ひずむ）** する。 |
| **表皮効果** | 高周波（高い周波数）の電気は、導体の**表面ばかりを通る**。 |

## 解説

### 1. 電磁気学の基本法則

電磁気の基本は、「左手と右手」「ファラデーとレンツ」をペアで分けるのが一番分かりやすい方法です。

- **左手は「モーター」、右手は「発電機」**
- **左手（モーター）**：すでに電気が流れているところに磁石を近づけると、ポンと「力」が生まれて動き出します。この「動く向き」を知るためのものです。
- **右手（発電機）**：磁石のそばで、自分の手で導体を「動かす」と、あとから「電気」が生まれます。この「電気の流れる向き」を知るためのものです。

- **ファラデーは「数字（大きさ）」、レンツは「あまのじゃく（向き）」**
- 磁石を動かして電気を作る（電磁誘導）とき、磁石をものすごいスピードで動かすと、生まれる電気は大きくなります。この「どれくらい大きいか」を決めるのが**ファラデーの法則**です。
- 逆に、磁石を近づけると「来るな！」と押し戻すような磁石の力をわざわざ作って、磁石が遠ざかると「行かないで！」と引き止めるような磁石の力を自分で作ります。このように、いつも変化の邪魔をしようとする「あまのじゃくな向き」を決めるのが**レンツの法則**です。


### 2. 各種電気現象のメカニズム

- **熱と電気のコンビ（ゼーベック・ペルチェ）**
  - 温度の差から電気を作るのが**ゼーベック効果**です。温度計などに使われます。
  - 逆に、電気を流して温度の差（片方を冷やす）を作るのが**ペルチェ効果**です。小型の冷蔵庫などに使われています。

- **ホール効果**
  - まっすぐ走っている電気（電子）の横から磁石の力をかけると、電子がフワッと横に曲げられてしまいます。すると、片方の壁に電子がたくさん溜まるので、横向きの電圧が生まれます。これがホール効果です。

- **表皮効果**
  - 無線で使うような「高周波」の電気は、ちょっぴり恥ずかしがり屋です。線の真ん中を通るのが苦手で、なぜか「線の表面ギリギリ」のところばかりに集まって流れていこうとします。そのため、高周波の線は真ん中が空っぽのパイプでも、表面さえきれいなら電気はしっかり流れます。

### グループ1：電気と磁気に関する法則

**問題 1 （記述の正誤判断判定：パターンA）**

次の記述は、電気と磁気に関する法則について述べたものである。このうち正しいものを1、誤っているものを2として解答せよ。

ア 直線状の導体に電流を流したとき、電流の流れる方向と導体の周囲に生じる磁界の方向との関係を表したものを、アンペアの右ねじの法則という。
イ 運動している導体が磁束を横切ると、導体に起電力が発生する。磁界の方向、磁界中の導体の運動の方向及び導体に発生する誘導起電力の方向の三者の関係を表したものを、フレミングの左手の法則という。
ウ 磁界中に置かれた導体に電流を流すと、導体に電磁力が働く。このとき、磁界の方向、電流の方向及び電磁力の方向の三者の関係を表したものを、フレミングの右手の法則という。
エ 電磁誘導によって生じる誘導起電力の方向は、その起電力による誘導電流の作る磁束が、もとの磁束の変化を妨げるような方向である。これをレンツの法則という。
オ 電磁誘導によってコイルに誘起される起電力の大きさは、コイルと鎖交する磁束の時間に対する変化の割合に比例する。これを電磁誘導に関するビオ・サバールの法則という。

**解法：**

- ア：正しい記述です。
- イ：導体を動かして生まれる電気の向きは、フレミングの**右手**の法則です。よって誤り。
- ウ：電流を流して生まれる力の向きは、フレミングの**左手**の法則です。よって誤り。
- エ：正しい記述です。変化を邪魔する（妨げる）のはレンツの法則です。
- オ：起電力の大きさが変化の割合に比例するのは、**ファラデー**の電磁誘導の法則です。よって誤り。

**✅正解：ア：1、イ：2、ウ：2、エ：1、オ：2**

**問題 3 （穴埋め組み合わせ問題）**

次の記述は、電気と磁気に関する法則について述べたものである。 $[\quad]$ 内に入れるべき字句の正しい組合せを下の番号から選べ。

(1) 電磁誘導によって生じる誘導起電力の方向は、その起電力による誘導電流の作る磁束が、もとの磁束の変化を妨げるような方向である。これを $[ A ]$ の法則という。
(2) 電磁誘導によってコイルに誘起される起電力の大きさは、コイルと鎖交する磁束の時間に対する変化の割合に比例する。これを電磁誘導に関する $[ B ]$ の法則という。
(3) 運動している導体が磁束を横切ると、導体に起電力が発生する。磁界の方向、磁界中の導体の運動の方向及び導体に発生する誘導起電力の方向が互いに直角な三者の関係を表したものを、フレミングの $[ C ]$ の法則という。

|  | A | B | C |
| --- | --- | --- | --- |
| **1** | レンツ | ファラデー | 右手 |
| **2** | アンペア | ビオ・サバール | 左手 |
| **3** | ビオ・サバール | レンツ | 右手 |
| **4** | ファラデー | アンペア | 左手 |

**解法：**

- (1) 変化を「妨げる方向」と言っているので、$A =$ **レンツ**。
- (2) 起電力の「大きさ」の話をしているので、$B =$ **ファラデー**。
- (3) 運動している導体に生まれる電気の話なので、$C =$ **右手**。
この組み合わせになっているのは **1** 番です。

**✅正解：1**

### グループ2：各種電気現象
